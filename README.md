## Galeria de Artes Online - Frontend

Este repositório contém a interface web (Frontend) do projeto de Galeria de Artes. O site consome a API do backend para exibir dinamicamente uma coleção de obras de arte.

### Aluno
* **Nome:** João Gois de Otoni - 202303038305
* **Disciplina:** Projeto Cloud

### Tecnologias 
* **Linguagem:** HTML5, CSS3 e JavaScript (ES6+)
* **Hospedagem:** Azure Static Web Apps
* **Integração:** Fetch API para consumo de dados do Backend

### Funcionalidades
* **Exibição Dinâmica:** Carrega os cards das obras automaticamente ao acessar a página.
* **Integração com Nuvem:** As imagens são carregadas diretamente do Azure Blob Storage e os textos do PostgreSQL (via Azure Function).
